# COVID-19 Dashboard Analysis (Power BI Project)

## Project Overview

This project presents an interactive Power BI dashboard analyzing COVID-19 data to track global trends, compare country-wise performance, and identify key patterns in cases, deaths, and recovery rates.

The objective is to transform raw data into actionable insights for better understanding of pandemic impact.

## Tools and Technologies

* Power BI
* Data Visualization
* Data Cleaning and Transformation (Power Query)
* DAX (Data Analysis Expressions)

## Dashboard Features

* Total confirmed cases, deaths, and recoveries
* Country-wise comparison of COVID-19 impact
* Time-series analysis of cases and deaths
* Recovery and mortality rate tracking
* Interactive filters for dynamic analysis

## Key Insights

* Identified countries with highest case growth rates
* Observed trends in recovery vs mortality rates over time
* Highlighted periods of rapid increase in infections
* Compared pandemic impact across different regions
